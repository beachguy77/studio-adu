# Aisling ADU

Portfolio site for ADU design and construction services in Santa Barbara County.

## Preview locally

```bash
cd /Users/giovanyreyes/Projects/aisling-adu
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

## Deploy

Push to GitHub and connect to [Netlify](https://netlify.com), [Vercel](https://vercel.com), or [Cloudflare Pages](https://pages.cloudflare.com) for free static hosting.

## Contact form

The contact form is a placeholder. When deploying:

- **Netlify**: add `netlify` attribute to the `<form>` tag
- **Formspree**: set `action="https://formspree.io/f/YOUR_ID"` on the form
